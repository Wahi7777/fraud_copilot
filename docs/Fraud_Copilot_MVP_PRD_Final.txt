Agentic Fraud Investigation Co‑Pilot — MVP
Version: Final Build Specification v5.0
Author: Sid Wahi
1. Product Overview
Banks and financial institutions receive large volumes of fraud alerts from transaction monitoring systems.
Each alert requires manual investigation including transaction history analysis, behavioural anomaly detection,
device/IP checks, beneficiary analysis, network relationships, fraud typology identification and documentation.
Typical investigation time: 10–30 minutes per alert.
2. Product Solution
The Fraud Investigation Co‑Pilot automates alert investigation.
Workflow:
Fraud Alert → Investigation Context → CrewAI Agents → Evidence Registry → Decision Agent → Risk Score → Dashboard.
The system assists analysts but does not replace fraud detection systems.

Included in MVP:

Excluded:
• Real bank integrations

• ML models
• Scenario generation engines
5. System Architecture
Datasets → Feature Builder → EnrichedTransactionRecord → InvestigationState
→ CrewAI Agents (Pattern, Behaviour, Network, Typology, Decision)
→ Evidence Registry → Fraud Score → Analyst Dashboard
6. Agent Autonomy Model
Agents operate with bounded autonomy.
They reason within a defined domain but cannot modify the workflow, spawn new agents,
or change scoring logic.
Agents run sequentially under CrewAI orchestration.

16. Investigation Agents
PatternAgent – velocity anomalies and transfer spikes
BehaviourAgent – device and location anomalies
NetworkAgent – graph analysis and account links
TypologyAgent – fraud pattern classification
DecisionAgent – final score and recommendation
17. Fraud Risk Scoring
Signal weights example:
HIGH_VELOCITY = 25
NEW_DEVICE = 20
NEW_COUNTRY = 15
LINKED_FRAUD_ACCOUNT = 30
Score = min(sum(weights)/100, 0.99)

Use pytest.

pipeline and API endpoints.

Code must be modular, include docstrings, avoid hardcoded paths,
handle malformed data and support Docker deployment.
24. Definition of Done

CrewAI agents generate structured evidence.
Risk score and recommendation returned.
Dashboard displays results.
All tests pass.
Docker deployment works.
